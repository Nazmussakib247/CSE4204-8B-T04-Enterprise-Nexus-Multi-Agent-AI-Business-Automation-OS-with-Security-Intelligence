List Candidates

POST
/api/v1/hr/candidates content_copy

Retrieve a paginated list of candidates analyzed by the HR Agent. Supports filtering by score, experience, and skills. This endpoint uses advanced vector search to return the most relevant candidates based on the role requirements.

QUERY PARAMETERS

Name Type Required Description
page Integer No The page number for pagination (defaults to 1).
limit Integer No Number of results per page (max 100).
role_id String Yes The unique identifier of the job role to filter against.

EXAMPLE REQUEST

curl -X POST \
  https://api.nexus.ai/v1/hr/candidates \
  -H 'Authorization: Bearer TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "role_id": "role_8821x",
    "min_score": 85
  }'

EXAMPLE RESPONSE

{
  "status": "success",
  "data": {
    "candidates": [
      {
        "id": "cand_01",
        "name": "Sarah Chen",
        "score": 94.2,
        "skills": ["Rust", "AI"]
      }
    ],
    "pagination": {
      "total": 124,
      "page": 1
    }
  }
}
info

AUTHORIZATION REQUIRED

This endpoint requires a valid API key with hr.read scope. Key rotation is recommended every 90 days.
